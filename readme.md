# README

By Kashif Bhatti.
Some of my Magento 2 sample code.

#### There are 5 extensions
1. AliDropShipping - web scraping from AliExpress for dropshipping products
2. HomeProducts - set up comma separated values of product IDs to insert into a carousel on the home page
3. ReferaFriend - Enter your and your friend's email address to receive a voucher
4. UltimateSearch - Ajax searching products using VueJS [test here](https://magento.preview1.co.uk)
5. Blog - Can add blog posts in admin and add comments from front-end
6. HomeCarousel - Add images hero carousel slider on the home page from admin
7. NewProducts - choose categories in admin to show new products with a category tab switcher on the home page
8. CustomerPhotos - Allow customers to upload their photos from front-end and add a homepage slider of customer photos
